# CreditDevice


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'credit_device'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install credit_device

## Usage

```
      CreditDevice.configure do |config|
        config.url = "url"
        config.username = "username"
        config.password = "password"
      end
```
